Penjelasan singkat pembuatan website sederhana :
1. Tentukan bahan yang akan dijadikan Website
2. Buka aplikasi VSCode atau Notepad++.
3. Pilih file >> New Text File >> Setelah muncul kata "untitled 1" pilih file kemudian save as
4. Masuk ke data C buat 1 folder baru khusus untuk website.
5. Atur Format penyimpanan menjadi Html >> Css >> js >> sesuai kebutuhan.
6. Kembali lagi ke VSCode dan susun scriptnya sampai bisa membentuk elemen yang siap di operasikan.
